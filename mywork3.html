William Shakespeare Sonnet Project

This project involves formatting and styling ten selected sonnets by William Shakespeare in a one-page HTML document with an external CSS stylesheet, including at least one external link, one internal link, one image insert, and one background image.

View Page

Overview

For this school coding project, I was tasked with creating a a project that involved formatting and styling ten selected sonnets by William Shakespeare in a one-page HTML document. To enhance the visual appeal and user experience, I incorporated an external CSS stylesheet, as well as a background image and image insert. Additionally, I included an external link to provide users with further resources and an internal link to facilitate navigation.
