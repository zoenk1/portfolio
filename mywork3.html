Overview

For this school coding project, I was tasked with creating a website for a recipe of my choice using CSS Grid. The focus of the project was to create a responsive design, which required me to explore new web technologies and develop my skills in web development and design. Throughout the project, I was able to showcase my creativity by designing a visually appealing website that effectively presented my chosen recipe. Overall, this project challenged me to push my limits and develop new skills while exploring the exciting world of web development.
